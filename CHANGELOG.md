# Changelog

## 0.1.1 - 2026-04-26

- 升级 Chrome 插件和 npm 包小版本号。
- 增加 Chrome 插件 PNG 图标声明和打包元数据测试。
- 补充 README，说明安装、更新、备份、权限和常见问题。
- 新增隐私说明文件。
- 新增 MIT License。
- 增强插件打包检查，减少分享 zip 时漏文件的风险。

## 0.1.0 - 2026-04-26

- 将原 Cloudflare Pages 网址导航迁移为本地 Chrome 新标签页插件。
- 移除管理员密码，改为本地普通编辑。
- 使用 `chrome.storage.local` 保存导航数据。
- 支持分组、网站卡片、搜索、编辑、删除和排序。
- 支持网站点击次数统计。
- 支持 JSON 和 iTab `.itabdata` 导入。
- 支持导出 JSON 备份和本地自动备份。
- 支持重复网址检测、批量整理和批量网址状态检测。
- 支持深色/浅色主题、卡片布局、背景和分组颜色。
- 支持首页快速新增分组和网站。
